# Ear Trainer

Interactive ear training app for intervals and triads.

**Features:**
- Phase 1: Interval recognition (Unison through Major 9th)
- Phase 2: Triad identification (Maj, Min, Dim, Aug, Sus2, Sus4, Maj7, Min7, Dom7)
- Real sampled instruments (Choir, Voice, Piano, Violin)
- Piano keyboard + guitar fretboard visualization
- Key selector, I-IV-V cadence, score tracking
- Reference sheet with song mnemonics and chord formulas

**Tech:** Pure HTML/JS, no dependencies. Audio samples embedded as base64.

## Deploy

This is a single `index.html` file. Host anywhere:
- GitHub Pages (enable in repo settings)
- Drop on any static host (Netlify, Vercel, Cloudflare Pages)
- Open locally in any browser
